## 2025-03-09 - Declarative Animations for Dynamic SVGs
**Learning:** Imperatively querying SVG elements with `querySelectorAll` and assigning inline styles during a `useEffect` inside React components causes unnecessary re-renders, layout recalculations, and desyncs. The length properties computed in `useEffect` can be calculated precisely during render and used directly via inline styles. Furthermore, dynamically modifying keyframes using inline CSS variables directly is cleaner and eliminates imperative CSS string mutation.
**Action:** When building interactive or animated SVG-based components in React, avoid `useEffect` for element dimension setup. Calculate dimensions (like line lengths) mathematically inside a `useMemo` block and pass them declaratively to the SVG elements via inline styles (using CSS variables if keyframes are involved).
