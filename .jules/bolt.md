## 2025-03-08 - Optimized NeuralNetworkDiagram Animation Styles
**Learning:** Found an anti-pattern in `NeuralNetworkDiagram.tsx` where visual length measurements for SVG line elements were being parsed using `querySelectorAll` and `getAttribute` inside a `useEffect`. This forces severe layout thrashing (reflows) because it reads from the DOM immediately after rendering, and then injects style mutations that cause further reflows. Furthermore, the animation keyframes incorrectly hardcoded a fixed offset (`100`), which causes visual glitches if the lines are longer or shorter.
**Action:** Always avoid querying the DOM to calculate dimensions if the data can be derived mathematically. Calculate structural values like SVG paths during the render phase in a `useMemo` block, and declaratively pass these values down to elements via CSS variables (`style={{ '--path-length': length }}`).
